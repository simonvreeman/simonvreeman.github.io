What makes this A/B calculator different?

The number one mistake in A/B tests is calling a test too early before enough data is collected.
This A/B split test calculator tries to solve this problem by - in addition to the regular significance level - calculating the recommended data sample size, statistical power and test duration as well.
Although the outcome of this calculator is mathematically correct, please bear in mind that this is only a part of the story and things can get a bit more complicated.

Our goal was to create a calculator which:

  • helps to answer the question "do I have enough data to stop my experiment?" in an easy-to-understand format
  • show both detailed statistical data for analysts, and clear verdicts in plain English for greenhorns
