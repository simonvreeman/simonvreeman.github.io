Un-personalized search in Google.nl

Search Operators: site: cache: info: inurl: intitle: filetype:

Current used search parameters

  • Turn personalised search off: pws=0
  • Set GEO location to the Netherlands: gl=nl
  • Set interface language to Dutch: hl=nl
  • Number of results in de SERP: num=10
  • Open links in a new tab: newwindow=1

To make sure the results are really un-personalized, use the private browsing mode of you internet browser or a private browser like WhiteHat Aviator.
