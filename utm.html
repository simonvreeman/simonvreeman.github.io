Google Analytics UTM Campaign URL Builder

Home | CRO Tools

A simple tool to create an URL with Google Analytics UTM campaign parameters for your manual tagging marketing campaigns. A couple of features; All input will be lowercase appended to the URL. The Google Analytics UTM Campaign URL Builder will check if the URL already contains parameters and if it returns a successful HTTP response status code [200]. With one click you can add the URL to your clipboard.

Make sure you are strict in using utm_source and utm_medium. Otherwise, you will end up with a lot of traffic in the (other) or unassigned Default Channel Groupings. More information about the Default Channel Grouping definitions and rules can be found here for Universal and here for GA4.
