EntityMap Custom Vocabulary v1

Namespace for Vreeman.com

This is the custom EntityMap v1.0 vocabulary namespace for vreeman.com, identified by the URI https://vreeman.com/entitymap/vocab/v1. It defines the site-specific terms — one type and one predicate — that extend the EntityMap core vocabulary and are referenced from the site's EntityMap document.

See the EntityMap itself: the machine-readable entitymap.json and the human-readable entitymap.html.

Custom type: Work

URI: https://vreeman.com/entitymap/vocab/v1#Work

A literary or textual work — a book, treatise, letter, essay, collection, or translation/edition. EntityMap v1.0's 16 core types include no creative-work type, so vreeman.com declares Work for the texts in its Stoicism library (the Meditations, the Discourses and Enchiridion, Seneca's Moral Letters, Stockdale's essays, etc.).

Custom predicate: TRANSLATED_BY

Subject
a Work
Object
a Person (the translator)

Asserts that the subject work is rendered into another language by the object person. Distinct from the standard AUTHORED_BY, which attributes original authorship. Example: the Meditations TRANSLATED_BY Gregory Hays.

Per EntityMap's custom-predicate rules, TRANSLATED_BY is written in uppercase and does not collide with any of the 24 standard predicates.
